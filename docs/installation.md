# Installation

## Requirements

- Python 3.10+
- Supported OS: Windows, Ubuntu, Arch Linux

## Clone Repository

```bash
git clone <repo-url>
cd <repo>
```

## Install Dependencies
```bash
pip install -r requirements.txt
```

## Environment Variables

Create a .env file:

SUPABASE_URL=...
SUPABASE_KEY=...
GOOGLE_CLIENT_ID=...

## Run the App
```bash 
python main.py
```
